# Трекер привычек и продуктивности

WPF-приложение (.NET 8) для отслеживания привычек, серий выполнения и задач. Ролевой доступ, in-memory хранилище, опциональный SQL Server backend.

## Запуск

```
dotnet run
```

или открыть `HabitTracker.csproj` в Visual Studio и нажать F5.

После структурных изменений делайте **Build → Rebuild Solution** (или `dotnet clean && dotnet build`), чтобы сбросить кэш XAML.

## Тестовые входы

| Логин | Пароль | Роль          |
|-------|--------|---------------|
| admin | admin  | Администратор |
| user  | user   | Пользователь  |

## Экраны

- **Сегодня** — привычки на текущий день с отметкой выполнения, сводка
- **Привычки** — добавление/удаление привычек, категории, частота, серии
- **Задачи** — задачи с приоритетом и сроком
- **Статистика** — сводные показатели, полосы серий и QR-код отчёта
- **Категории** *(админ)* — общие категории привычек
- **Пользователи** *(админ)* — управление учётными записями

## Роли

- **Администратор (0)** — все экраны, включая управление категориями и пользователями
- **Пользователь (1)** — личные привычки, задачи, статистика

Сайдбар строится динамически по роли в `MainWindow.BuildNav`.

## База данных

`Database/HabitTracker.sql` — полная схема для SQL Server (таблицы, внешние ключи, ограничения, индексы, тестовые данные). Приложение при старте подключается к SQL Server через ADO.NET (`Microsoft.Data.SqlClient`) и загружает пользователей, категории, привычки, отметки и задачи из этой базы.

По умолчанию используется строка подключения:

```
Server=(localdb)\MSSQLLocalDB;Database=HabitTracker;Trusted_Connection=True;TrustServerCertificate=True
```

Чтобы указать другой SQL Server, задайте переменную окружения `HABITTRACKER_CONNECTION_STRING`. Если подключение недоступно, приложение автоматически запускается на демо-данных в памяти, чтобы интерфейс оставался рабочим.

## Архитектура

- Code-behind WPF, синглтон `DataStore`, `ObservableCollection`
- Тёмная палитра индиго/фиолетовый + мятный акцент в `App.xaml`
- Серия (streak) считается в `DataStore.RecalcStreak` по последовательным дням с отметкой
